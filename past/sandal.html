About the Project

I was contacted by a local realtor who was looking to have someone develop a website for them. They contacted me, and this was my first experience working with a client.
While working with this client, I was able to utilize bootstrap with the website, and help build them a responsive website that help them get their information out. I was able to connect the website to various Zillow tools and help pull important realty data via their API.
Unfortunately, the owner no longer uses the site since they decided to change their career path, but the website files are currently hosted on this server so that people can view my previous work.
