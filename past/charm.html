About the Project

My friend Sebastian Dang and I were hired to help a group of individuals create a mobile social networking app. For the first milestone, they wanted us to create a very simple prototype, so we created a very basic android login page, which has server sided authenication with a database.

Unfortunately, the application did not proceed after the first milestone, but we were still able to release what we learned during a short time working on the application.
