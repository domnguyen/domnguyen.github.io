Discord Bot

A .NET Discord Bot.

URL : https://github.com/domnguyen/Discord-Bot

Technology Used:

  • C# / .NET
  • ffmpeg
  • Visual Studio 2017

About the Project

Discord is a VoIP application that is catered to gaming communities. Discord allows users to create their own server, and invite their friends or players around the world to join their server. In a server I was involved in, I wanted to create my own bot to both learn about C#, and as well as control everything the bot did. As a result, this bot was born.
This bot currently is in early release stage, as it has basic response commands integrated as well as a music player function. By January 2018, the bot should be ready for public release.
